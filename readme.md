## Caracteres Especiales
Deben contener una barra invertida delante para ser visibles
>\\  barra invertida  
\`  acento invertido  
\*  asterisco  
\_  guión bajo  
\{\} llaves  
\[\] corchetes  
\(\) paréntesis  
\#  almohadilla  
\+  símbolo de suma  
\-  guión  
\.  punto  
\!  exclamación
